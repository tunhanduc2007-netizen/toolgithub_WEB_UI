🔑 Hướng dẫn đăng nhập GitHub CLI

Nếu trạng thái GitHub login báo CHƯA ĐĂNG NHẬP, hãy mở terminal trên máy tính của bạn và chạy lệnh sau:

Làm theo hướng dẫn trên terminal để liên kết tài khoản GitHub của bạn với máy tính này.
